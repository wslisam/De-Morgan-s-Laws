# De Morgan's Laws

Welcome to the interactive learning module for De Morgan's Laws! This single-page application is designed specifically to help students visualize and internalize one of the most critical concepts in Boolean Algebra.

## 🎯 Pedagogical Goals

Intuitive Understanding: Move away from pure truth-table memorization by using relatable, real-world analogies (e.g., ordering noodles).

Visual Logical Flow: Trace how a signal changes as it passes through each Logic Gate.

Critical Thinking: Formulate your own expressions before revealing the answers in the Self-Assessment module.

## 🚀 How to Use This App

Read the Analogy: Start with the "Noodle Analogy" to understand the core logic conceptually.

Experiment with the Simulator: - Toggle the inputs for A (Spicy) and B (Hot).

Watch how the True (Green) and False (Gray) signals propagate through Circuit 1 and Circuit 2.

Observe that regardless of the input combination, both final outputs are always identical, practically proving De Morgan's Law.

Reflect on Misconceptions: Read the common pitfalls to ensure you aren't making typical distribution errors.

Take the Mini-Quiz: Use the text area to draft your own logical expression based on the smart home scenario. Only click 'Compare with Answer' once you have thought it through!

## 💡 Self-Directed Learning Tips

Try mapping other real-world scenarios to Boolean logic (e.g., "I will play games if it's raining AND I have finished homework").

Toggle the language to English/繁體中文 in the top right corner to familiarize yourself with bilingual technical terms (e.g., Logic Gates, Boolean Algebra).
